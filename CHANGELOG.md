# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2025-12-14

### Added

- **Icon Insertion Dialog**: Added a new command `Insert Mermaid Icon` (Hotkeys: `Cmd+Shift+i`) to easily search and insert icons into the editor.
- **Icon Browser in Settings**: Added a "Supported icons" section in the plugin settings to browse and filter available icons.
- **New Icon Set Support**: Added support for `lucide` icons in addition to `logos`.
- **Linting & Formatting**: Added ESLint and Prettier configuration for code quality and consistency.
